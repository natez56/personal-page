# Notes Taken From https://www.tutorialspoint.com/html5

## General HTML5 Changes

- Uppercase tag names.
- Quotes are optional for attributes.
- Attribute values are optional.
- Closing empty elements are optional.

## The DOCTYPE

```html
<!DOCTYPE html>
```

## Character Encoding

```html
<meta charset="UTF-8">
```

## <script> tag

```html
<script src="script.js"></script>
```

## <link> tag

```html
<link rel="stylesheet" href="stylefile.css">
```

## Example HTML5 page:

```html
<!DOCTYPE html>

<html>
   <head>
      <meta charset="utf-8">
      <title>...</title>
   </head>

   <body>
      <header>...</header>
      <nav>...</nav>

      <article>
         <section>
            ...
         </section>
      </article>
      <aside>...</aside>

      <footer>...</footer>
   </body>
</html>
```
